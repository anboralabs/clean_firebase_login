# Clean Login Firebase

Clean Login Firebase is a clean code implementation of login with firebase server. it is only demo how to implement it without boilerplate code.

## Usage

```
git clone https://github.com/anboralabs/clean_firebase_login.git
```
Open it with Android studio.
